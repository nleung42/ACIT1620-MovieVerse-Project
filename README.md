# MovieVerse – Film Review Showcase

## Overview
This starter project is part of BCIT ACIT 1620 – Fundamental Web Technologies.  
Build a multi-page film review website using semantic HTML and CSS transitions.

## File Structure
```
MovieVerse_Project/
├── index.html
├── reviews.html
├── contact.html
├── css/
│   └── style.css
└── images/
```

## Setup
1. Extract the ZIP file.
2. Open in VS Code.
3. Edit HTML and CSS to customize your content.
4. Add movie posters and visuals.
5. Test responsiveness on different devices.

## Submission
Compress and submit as `MovieVerse_YourName.zip`.
